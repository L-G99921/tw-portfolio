# BillSense AI Overview

BillSense AI is an intelligent energy analysis tool that helps customers understand their energy consumption. It connects to the utility smart meter and provides detailed bill breakdowns, usage insights, and personalized savings recommendations.

## Key Features

- Bill analysis with month-over-month and year-over-year comparisons
- Energy usage breakdown by category (heating, cooling, and appliances)
- Personalized savings recommendations based on usage patterns
- Neighborhood comparisons against similar homes
- Rate plan optimization suggestions
- Historical billing data and trends

## Limitations

- Usage data excludes taxes, fees, and voluntary program charges
- Displayed kilowatt-hour (kWh) values are estimates and may vary slightly from official bill statements
- Recommendations require accurate home profile information to function correctly
- Customers on a fixed-term rate plan agreement cannot switch plans until the agreement expires

## Interface Overview

The BillSense AI interface displays the customer's service address in the header. Use the left sidebar to navigate between sections.

### Main Sections

| Section | Description |
|---------|-------------|
| Insights | Bill summary, usage changes, and personalized recommendations |
| Usage | Detailed breakdown by category and daily patterns |
| History | Historical billing data organized by year |
| My Home | Home profile and appliance configuration |
| My Plan | Current rate plan and available alternatives |
| Energy Usage Data | Link to the utility's external energy data portal, where customers can view and download detailed meter usage records |

## Glossary

| Term | Definition |
|------|------------|
| Always On | Energy consumed by devices in standby mode |
| Disaggregation | Breaking down total usage into individual appliance categories |
| kWh | Kilowatt-hour — the standard unit of energy consumption |
| Off-Peak | Hours when electricity rates are lower |
| On-Peak | Hours when electricity rates are higher |
| Rate Plan | The pricing structure used to calculate the customer's electricity bill |
| Time of Use (TOU) | A rate plan type where prices vary based on the time of day energy is consumed |
